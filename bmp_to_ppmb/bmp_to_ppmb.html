BMP_TO_PPMB
BMP => binary Portable Pixel Map (PPM) Conversion

BMP_TO_PPMB is a C++ program which converts a Microsoft BMP file to a binary Portable Pixel Map (PPM) file.

Usage:

bmp_to_ppmb file.bmp file.ppm
reads data from file.bmp and creates an equivalent graphics file in file.ppm.
