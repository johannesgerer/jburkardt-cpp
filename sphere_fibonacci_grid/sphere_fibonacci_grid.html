SPHERE_FIBONACCI_GRID
Fibonacci Spiral Grid on a Sphere

SPHERE_FIBONACCI_GRID is a C++ library which constructs a grid of points using the Fibonacci spiral over the surface of a sphere in 3D.

Languages:

SPHERE_FIBONACCI_GRID is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version and a Python version.
