DISK_MONTE_CARLO
Monte Carlo Estimate of Integrals in the Unit Disk

DISK_MONTE_CARLO is a C++ library which uses the Monte Carlo method to estimate the integral of a function over the unit disk in 2D.

Languages:

DISK_MONTE_CARLO is available in a C version and a C++ version and a FORTRAN77 version and a FORTRAN90 version and a MATLAB version.
